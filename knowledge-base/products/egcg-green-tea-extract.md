# EGCG (Green Tea Extract)

**Category:** supplement
**Evidence Grade:** B
**Risk Profile:** low

## Description
Potent autophagy inducer via multiple pathways

## Mechanisms of Action
- Autophagy induction
- Antioxidant
- Anti-inflammatory
- AMPK activation

## Dosage
- **Standard:** 400-500mg/day
- **Range:** 200-800mg/day
- **Notes:** Take on empty stomach for better absorption. Avoid with iron-rich meals.

## Key Findings
- Potent autophagy inducer via multiple pathways
- Reduced all-cause mortality in large cohort studies
- Cancer-preventive properties in multiple cancers
