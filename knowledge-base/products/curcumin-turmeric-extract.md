# Curcumin (Turmeric Extract)

**Category:** supplement
**Evidence Grade:** B
**Risk Profile:** low

## Description
Potent NF-κB inhibitor — master inflammation regulator

## Mechanisms of Action
- Anti-inflammatory (NF-κB)
- Antioxidant
- Senomorphic
- Neuroprotective

## Dosage
- **Standard:** 500-1000mg/day (enhanced absorption form)
- **Range:** 200-2000mg/day
- **Notes:** Bioavailability is poor — use Longvida, Meriva, or with piperine. Fat-soluble.

## Key Findings
- Potent NF-κB inhibitor — master inflammation regulator
- Improved memory and attention in BDNF-related study
- Senomorphic: reduces SASP without killing senescent cells
