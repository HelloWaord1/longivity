# Omega-3 (EPA/DHA)

**Category:** supplement
**Evidence Grade:** A
**Risk Profile:** low

## Description
Strong evidence for cardiovascular benefit

## Mechanisms of Action
- Anti-inflammatory
- Cardiovascular
- Neuroprotective
- Cell membrane integrity

## Dosage
- **Standard:** 2-4g/day (combined EPA+DHA)
- **Range:** 1-5g/day
- **Notes:** Higher EPA:DHA ratio for inflammation. Test omega-3 index (target 8-12%).

## Key Findings
- Strong evidence for cardiovascular benefit
- Higher omega-3 index associated with longer telomeres
- VITAL trial: reduced heart attack risk by 28%
