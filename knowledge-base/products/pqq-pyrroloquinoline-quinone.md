# PQQ (Pyrroloquinoline Quinone)

**Category:** supplement
**Evidence Grade:** C
**Risk Profile:** low

## Description
Stimulates mitochondrial biogenesis via PGC-1α

## Mechanisms of Action
- Mitochondrial biogenesis
- Neuroprotective
- Antioxidant
- NGF stimulation

## Dosage
- **Standard:** 20mg/day
- **Range:** 10-40mg/day
- **Notes:** Synergistic with CoQ10. Found in natto and green tea.

## Key Findings
- Stimulates mitochondrial biogenesis via PGC-1α
- Improved cognitive function in elderly (Japanese trial)
- Stimulates nerve growth factor (NGF)
