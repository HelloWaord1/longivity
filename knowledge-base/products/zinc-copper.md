# Zinc + Copper

**Category:** supplement
**Evidence Grade:** A
**Risk Profile:** low

## Description
Essential for immune function — declines with age

## Mechanisms of Action
- Immune function
- Enzyme cofactor
- DNA repair
- Antioxidant (SOD)

## Dosage
- **Standard:** Zinc 15-30mg + Copper 1-2mg/day
- **Range:** Zinc 10-50mg
- **Notes:** ALWAYS pair zinc with copper — zinc depletes copper. 10:1 ratio zinc:copper.

## Key Findings
- Essential for immune function — declines with age
- Zinc required for DNA repair enzymes
- Copper essential for SOD (superoxide dismutase) — key antioxidant enzyme
